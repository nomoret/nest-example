# NestJS 공부용

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository. <br>
nestjs 프레임워크를 공부하기 위해 만든 레포

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
